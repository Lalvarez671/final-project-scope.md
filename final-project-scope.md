# Final Project Scope Statement 

## Project Purpose 
The Purpose of this project is to display my level of mastery over the skills I have learned and practiced through designing a developing a personal fitness focused website that showcases my workout journey, workout routines, and progress.
 
## Project Description 
This project involves designing and developong a professional and personal porfolio website that showcases my skills. The goal is to creat a clean, well organized, and professional website that can be shared with others. 

## Desired Results
- Create a fully functioning website using HTML and CSS
- Published website on a live URL using GitHub
- README file documenting the project
- Make sure the website is public, visually appealing, and accurate
- Organize and present workout routines and fitness content clearly

## Exclusions 
- Personalized coaching
- The Intergration of wearable fitness devices such as Apple watches.
- Developing a full fitness application
