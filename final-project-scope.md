# Final Project Scope Statement 

## Project Purpose 
The Purpose of this project is to display my level of mastery over the skills I have learned and practiced through designing a developing a personal fitness focused website that showcases my workout journey, workout routines, and progress.
 
## Project Description 
This project involves designing and developong a professional and personal porfolio website that showcases my skills. The goal is to creat a clean, well organized, and professional website that can be shared with others. 

## Desired Results
- Create a fully functioning website using HTML and CSS
- Published website on a live URL using GitHub
- README file documenting the project
- Make sure the website is public, visually appealing, and accurate
- Organize and present workout routines and fitness content clearly

## Exclusions 
- Personalized coaching
- The Intergration of wearable fitness devices such as Apple watches.
- Developing a full fitness application
- Medical or nutritional advice 

## Priorities  
The main priority of this project is to create a functional and visually appealing fitness website using HTML and CSS. Focus will be placed on keeping the design clean, organized, and easy to navigate.   

## Time  
This project will be completed within the course timeline. I will manage my time by breaking the project into smaller steps such as planning, development, testing, and publishing. This will help ensure the project is completed on time and meets all requirements.  

## Budget  
There is no financial cost required for this project since all tools used, such as HTML, CSS, and GitHub, are free. The main investment will be the time and effort needed to design, build, and refine the website.  

## Acceptance Criteria  
- The website must be fully functional and accessible through a live URL   
- The layout must be clean, organized, and visually appealing  
- Fitness content must be clearly presented  
- A README file must be included
